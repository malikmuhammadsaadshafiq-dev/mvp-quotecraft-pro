## Problem

Freelancers consistently undercharge or underestimate project scope leading to financial losses and scope creep.

## Who Is This For?

Independent contractors and small creative agencies

## Features

- **Add modular task templates with time estimates and prerequisite dependencies**
- **Submit branded proposals to clients with digital signature acceptance and revision tracking**
- **Track estimation accuracy comparing quoted hours vs actual logged time per project phase**
- **Calculate dynamic pricing based on complexity multipliers, rush fees, and historical data**

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/malikmuhammadsaadshafiq-dev/mvp-quotecraft-pro.git
cd mvp-quotecraft-pro
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
